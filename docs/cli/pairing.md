# `opensoul pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
opensoul pairing list whatsapp
opensoul pairing approve whatsapp <code> --notify
```
