# `opensoul setup`

Initialize `~/.opensoul/opensoul.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
opensoul setup
opensoul setup --workspace ~/.opensoul/workspace
```

To run the wizard via setup:

```bash
opensoul setup --wizard
```
