# `opensoul tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
opensoul tui
opensoul tui --url ws://127.0.0.1:18789 --token <token>
opensoul tui --session main --deliver
```
